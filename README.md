# Antariksh Sathi

Antariksh Sathi is the Aapka-Sathi family app for:

- ISRO mission watch
- 2026 sky events and meteor showers
- India stargazing spots by state
- global space agencies directory
- a student-friendly space curiosity tool

It is built as a static, offline-first web app with preloaded content packs, so the experience stays fast even without a live API.
